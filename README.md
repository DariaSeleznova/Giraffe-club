# 🦒 Giraffe Club

Responsive website for the **Giraffe Club language school** created as a frontend practice project.

This project demonstrates building a modern frontend layout using **SCSS, Webpack, and responsive design techniques**.

## ✨ Features

- responsive layout (mobile / tablet / desktop)
- mobile burger navigation
- modal form window
- interactive carousel
- responsive image gallery
- SCSS architecture (variables, mixins, maps)
- project bundling with Webpack

## ⚙️ Installation and Setup

Clone the repository:
git clone https://github.com/DariaSeleznova/Giraffe-club.git


Navigate to the project folder: cd Giraffe-club


Install dependencies:  npm install


Run development server:  npm run start


Build production version:  npm run build
